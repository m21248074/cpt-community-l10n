這是一個由社群發起的 **Cisco Packet Tracer (CPT)** 國際化翻譯計畫。我們的目標是提供最準確、最完整的繁體中文（及其他語言）介面，讓學習網路技術不再受到語言隔閡的限制。

---

## 📢 參與翻譯 (How to Contribute)

我們使用 **Crowdin** 作為協作平台，你不需要具備 Git 或程式背景，只要註冊帳號即可在瀏覽器上參與翻譯！

👉 **[點我加入 Crowdin 翻譯團隊](https://crowdin.com/project/packet-tracer-community-translation)**

> [!IMPORTANT]
> **請勿直接在 GitHub 提交 Pull Request 修改 `.ts` 檔案。**
> 所有的翻譯修改都應在 Crowdin 平台上完成，系統會自動將通過審核的譯文同步回此儲存庫。

---

## 🛠️ 安裝說明 (Installation)

如果你想在自己的 Packet Tracer 中使用目前的翻譯成果：

1. 前往本專案的 [Releases](../../releases) 頁面下載最新的 `.qm` 檔案。
2. 將下載的檔案複製到 Packet Tracer 的安裝路徑：
   - **Windows:** `C:\Program Files\Cisco Packet Tracer X.X\languages`
   - **Linux:** `/opt/pt/languages`
3. 開啟 Packet Tracer，進入 **Options > Preferences > Interface**。
4. 在 **Languages** 清單中選擇該語言，並點擊 **Change Language**。
5. 重啟程式即可生效。

## 📅 更新維護

由於 CPT 版本更新頻繁，當官方發布新版本時：
1. 我們會將最新的 `template.ts` 上傳至此 Repo。
2. Crowdin 將自動偵測新字串並開放翻譯。
3. 歡迎大家關注此專案以獲得最新進度。
